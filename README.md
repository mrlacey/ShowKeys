# ShowKeys

Show the default keyboard shortcuts on screen when comon commands are invoked.

Intended for use during code demos or presentations to help the audience see what you're typing.

Works with
- Cut
- Copy
- Paste
- Tab
- Back Tab
- Escape
- Delete
- Page Up
- Page Down
- Comment Selection
- Uncomment Selection
- Duplicate Selection
- Format Document
- Format Selection
- Insert Snippet
- Invoke Quick Info
- Move Selected Lines Down
- Move Selected Lines Up
- Undo
- Redo
- Rename
- Save
- Select All
- Surround With
